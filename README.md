# sms-spam-detector
Machine Learning project to detect spam SMS messages
Project Overview

This project is a Machine Learning model that classifies SMS messages as Spam or Ham (Not Spam) using Natural Language Processing.

📊 Dataset

SMS Spam Collection Dataset

5572 messages

Labels: ham / spam

🧠 Model Used

Multinomial Naive Bayes

Feature Extraction: CountVectorizer (Bag of Words)

🎯 Results

Accuracy: 98.39%

⚙️ How it works

Text messages are converted into numbers using CountVectorizer

Model learns patterns from spam and ham messages

It predicts whether new messages are spam or not

🚀 How to run

Open notebook

Run all cells

Test with custom messages

🛠 Tools Used

Python, Pandas, Scikit-learn, NLP
